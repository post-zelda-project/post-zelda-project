Features

Fully Open-Sources

This project is fully open-source, all of the source code posted at our github account.
You can download the lastest version source code and compiled DLL for lastest CryENGINE without any limitation.

Easy to customize

The project simpified many API of CryENGINE and define as a new interface, also we have many tutorials about how to use this project.
So that you can customize this project easily and use it as the foundation of your game project. For more tutorial, please watch about The YouTube channel.

Free To Use

As this project is a fan-art project, and limited by the licecnse, so this project will never charge any fees.
It means you only need to follow the license and condition of this project, then you can use this project freely.
